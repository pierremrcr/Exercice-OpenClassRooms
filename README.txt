# Exercice-OpenClassRooms
Ce repository répond à l'exercice sur les deux premières parties du cours sur Git et Github
L'exercice consiste à créer un Repository Git simple contenant ce fichier README.txt ainsi que deux autres fichiers minimum. Il faut ensuite créer 4 commits distincts montrant l'ajout et la modification de certains fichiers.
Une fois ce repository crée, il faut l'envoyer sur Github et renseigner l'URL du Repository Github dans le document texte fourni pour l'exercice puis compresser ce document dans un fichier .zip
